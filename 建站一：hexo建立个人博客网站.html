建站一：hexo建立个人博客网站

发表于 2016-09-24   |  

大家好，今天给大家分享一下自己建立个人博客网站的经验
本人在建立blog的过程中，在网上查阅了许多资料，费尽了10多天才成型
本期主要分享的是使用hexo+GitHub搭建一个自己的个人博客网站
由于本人也是个菜鸟，很多东西不能说出个所以然，如有错误还请各位指点

软件准备

Git

选择对应你系统版本的Git，安装一路默认即可

Node.js

建议下载LTS版本的Node.js，安装也是一路默认即可

Hexo

注意这个不是在网页上下载的
安装方式：

  1. 鼠标右键，点击Git Bash Here
  2. 输入npm install hexo-cli -g
    这样安装就完成了。

GitHub配置

  1. 进入官网GtiHub，注册登录，然后点击New repository
  2. 出现以下界面
    仓库名必须是yourname.github.io即此处应为Lee-rui.github.io，后面的可以默认，点击Create repository
