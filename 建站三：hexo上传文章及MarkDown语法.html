建站三：hexo上传文章及Markdown语法

发表于 2016-10-03   |  

本期主要跟大家分享的是如何上传文章以及Markdown语法

上传文章

在git bash下输入
$ hexo new [layout] <title>

布局layout

布局 路径
post source/_posts
page source
draft source/_drafts

对应文章，页面，草稿

例：
$ hexo new post 我的第一个页面

此时就会在_posts文件夹下生成一个 我的第一个页面.md文档

该文档需要使用Markdown语法编辑，即hexo上的文档都支持Markdown语法格式的文章。

文章编辑好后可以通过

部署到github上

Markdown语法

Markdown简介

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。

常用语法

标题

Markdown支持两种标题语法，Setext和atx

Setext以底线的形式，以两个数量以上的=和-。

atx以#加空格的形式，一共有6阶标题（即视#的数目而定）

引用
